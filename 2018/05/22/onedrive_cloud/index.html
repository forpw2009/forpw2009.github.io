【学习笔记】利用微软5T空间搭建私有云

2018-05-22 阅读量

最近闲来无事，看到TG群一个大佬共享一个云盘，私有的，简洁清爽，

于是虚心学习了一波，自己建成功喜欢的小伙伴也可以自己搭建，

完全免费，往下看教程就好啦o(^▽^)o

搭建效果链接 forpw2009.tk

功能

  1. 免费申请5T微软onedrive空间
  2. 不用服务器空间，不走服务器流量
  3. onedrive目录，文件分享直链下载
  4. 支持自定义域名，（微软不倒闭）永久有效

申请微软onedrive5T空间

  • 注册教育邮箱(有edu邮箱小伙伴可以跳过）

    访问 http://qqu.me:3000 自己注册一个临时邮箱，用于接收稍后微软发送发送的验证码

  • 注册微软365学生版

    邮箱注册好后，保持邮箱页面打开，新建浏览标签访问微软365进行注册，

    链接如下：https://signup.microsoft.com/signup?sku=student 输入临时邮箱进行注册

    自定义临时时邮箱

    输入自己定义的临时邮箱

输入get微软验证码

  • 登陆office.com

    输入刚刚注册等账号和密码，登陆即可暂时获得1T空间

  • 登陆桌面版，会自动获得另外4T空间，共计5T空间

    WIN10 系统直接打开自带onedrive客户端，登陆即可


申请免费域名，freenom网站进行申请

  • freenom是全球有名的域名供应商，可以免费申请.tk ；.ml 等相关域名

  • 注册后输入自己喜欢的字符串，进行check，既可以看到是否可以申请


  • 选择free的进行确认，记得将period改为一年，同样免费（到期后重新购买即可）


免费域名使用cloudflare进行托管

  • cloudflare是全球有名的域名托管商，可以托管免费申请的域名

  • 免费注册后，点击add site 进行添加域名

  • 安装要求，返回freenom网页进行设置，添加cloudflare接管的站点

  • 添加成功后，过1个小时左右，就可以在cloudflare网站看到自己域名被接管了

申请免费PHP空间，搭建网盘网页版

  • 免费PHP空间，度娘一下很多，在这推荐一个免费网站暮想云
    用邮箱免费注册，即可获得一年免费PHP空间，到期可以重新注册，永久免费


  • 下载github项目文件，地址

  • 将压缩包上传PHP空间，参考以下图片



  • 访问自己申请的域名即可打开网页进行网盘绑定设置，登陆申请的office365账号即可

小结

  • 利用微软提供的office365 学生优惠，申请免费的onedrive云空间

  • 寻找免费PHP空间搭建网盘网站框架，与onedrive进行对接

  • 绑定自己申请的域名，成功搭建自己私有云
