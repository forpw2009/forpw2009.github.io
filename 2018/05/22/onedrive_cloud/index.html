利用微软5T空间搭建私有云

2018-05-22 PV:

最近闲来无事，看到TG群一个大佬共享一个云盘，私有的，简洁清爽，于是虚心学习了一波，自己建成功
喜欢的小伙伴也可以自己搭建，往下看教程就好啦o(^▽^)o

功能

  1. 免费申请5T微软onedrive空间

  2. 不用服务器空间，不走服务器流量

  3. 直接列onedrive目录，文件直链下载

申请微软onedrive5T空间

  • 注册教育邮箱(有edu邮箱小伙伴可以跳过）

    访问 http://qqu.me:3000 自己注册一个临时邮箱，用于接收稍后微软发送发送的验证码

邮箱注册好后，保持邮箱页面打开，新建浏览标签访问微软365进行注册，链接如下：[https://signup.microsoft.com/signup?sku=student](https://signup.microsoft.com/signup?sku=student) 输入临时邮箱进行注册

  • 二维码扫描

    分享的节点以二维码显示，此时我们可以先保存到相册，在用Shadowrocket 扫描导入该节点

  • 自主添加服务器信息

    Shadowrocket 支持的协议模式还是很多的，这里我们以ssr为例

    最后不要忘了点击完成哦

  • 订阅节点
    由于这款工具本身并不自带节点，需要自己寻找有关信息，在此，博主为大家送上订阅地址(免费机场)，至于速度及延迟，大家自行测试

    别急哈，订阅地址在文章末为大家送上，先看完教程哦

延迟测试

首页有测试延迟的标点，点击即可，列表会显示每个节点的延迟，数值越小越好。

常用全局路由

  • 代理 ( 使用全局代理访问网络，通俗来说，手机访问任何网络都使用了你节点的ip )
  • 配置 ( 按照配置文件中的规则访问网络,具体看规则，可访问国内网站时直连，访问国外时用节点,也可去广告,屏蔽影视视频的开头广告如爱奇艺，腾讯视频等等 )

配置文件

给大家分享一个配置文件，可去广告，使用的时候，全局路由选择配置，长按拷贝链接
